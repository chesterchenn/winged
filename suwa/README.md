速蛙签到脚本

注册链接：[速蛙云](https://i.753d.top/kJFZ)

## 配置

环境变量

- SU_WA: 速蛙云的 AuthorizationMweb 请求头字段
- SU_DAY: 速娃云的重置日期
- PUSK_KEY: Server 酱的 sendkey

## 列表

- ✅ 统计每日签到流量
- ✅ 统计每月签到总流量

## 推送文本

成功签到推送的示范文本

```plain
签到成功，获得了 210MB 流量.
流量剩余 16.00GB
当月累计签到总计 1.45GB
```

重复签到推送的示范文本

```plain
您似乎已经成功签到过了...
流量剩余 16.00GB
当月累计签到总计 1.45GB
```

签到失败推送的示范文本

```plain
401 Unauthorized
```
